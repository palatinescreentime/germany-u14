📱 Getting Started

Add the app to your iPhone home screen so it opens full-screen with no browser bar:

1Open germany.bargraph.city in Safari
2Tap the Share button (box with arrow) at the bottom
3Scroll down and tap Add to Home Screen
4Tap Add — the G icon appears on your home screen
Parent view
Share germany.bargraph.city/?view=parent with parents. They see a tile for every game — tap any game for the live score, countdown clock, formation with player names, and match events. Use the QR button on the Live tab to share instantly on the sideline.
📋 Before Game Day
Roster & Team Name
  • Tap Roster from the season screen
  • Tap any player row to edit name, grade, foot, positions, or goalie willingness
  • Tap + Add Player to add  ·  tap × to remove
  • Team name is editable at the top of the Roster page
Setting Up a Game
  • Tap any game card from the season screen
  • Setup tab — enter opponent, date, location (auto-saves on exit)
  • Toggle availability — tap a name to mark OUT (red/strikethrough)
  • Coach Notes — type tactical notes and tap Save Notes
  • Status updates automatically: Upcoming → Live when clock starts → Complete when final score submitted
Building the Lineup
  • Tap Lineup tab → choose First Half or Second Half
  • Pick a formation: 4-3-3, 4-4-2, 4-2-3-1, 4-5-1, 3-5-2 — each half independent
  • Tap a player in the Player Pool at the bottom, then tap a position slot to place them
  • Tap any placed player to select, tap another to swap
  • Tap the arrow label below each bench player to set their planned sub target — shows as a gold outline on that player on the Live pitch
  • Copy from game imports any previous lineup including bench and sub targets
  • Tap Print for a printable sideline sheet
H2 rule
All six H1 bench players should start H2. Gold-bordered chips show who is new to the pitch that half.
⏱ Game Day — Live Tab
Clock
▶Start at kickoff — sets status to Live automatically
⏸Pause (halftime, stoppage)
±Adjust — type the actual minute if you forgot to start on time
↺Reset to 35:00 for the new half
QRShow QR code — parents scan to open the parent view

Counts down from 35:00. Gold color = overtime. Parents see the same clock.

Reading the Pitch
  • Players shown in actual positions, updating after every sub
  • Gold outline = a bench player is planned to sub for them — coming off soon
  • No outline = no sub planned yet
  • Dark red + ⚠ = marked unavailable but on the pitch
Sub planning
Set sub targets on the Lineup page. Gold outlines appear automatically on the Live pitch for each targeted player — see who is coming off at a glance.
Making Substitutions
  • Tap any player chip on the pitch — pre-selects them to come OFF
  • Tap a bench player name — pre-selects them to come ON
  • Tap the arrow label under a bench player to update their target during the game
  • Clock minute auto-fills — edit if needed, then tap Confirm Sub
  • Incoming player takes the correct position — gold outline clears
Tracking the Score
  • Tap Goal to log scorer and optional assist — Germany score increments automatically
  • For opponent goals: tap the + button under the opponent score — instant, no typing
  • Tap − to correct an accidental opponent goal entry
  • Both scores update on the parent view immediately
  • Edit either score manually using the number input fields at any time
Notes
  • Tap Note to log a formation change or tactical observation with a timestamp
✅ After the Game
1Tap Submit Final Score on the Live tab — unlocks Report, marks game complete
2Report tab — playing time bars, goals & assists, full match log
3Tap Bag Tags from the season screen to award tags
4Hardest Worker alert shows who still needs that tag this season
Clean Sheet
If Germany kept a shutout, a banner on the Bag Tags page awards Clean Sheet to the whole team in one tap.
👥 Parent View
  • URL: germany.bargraph.city/?view=parent
  • Shows a tile for every game — tap to view formation, score, clock, and events
  • Pitch updates as subs are made in real time
  • Score updates instantly when you log a goal or tap + for an opponent goal
  • Auto-refreshes every 20 seconds — parents do nothing
🧱 Bag Tags
  • Award Tags — select game, tap chips to award. Strikethrough = already received this season.
  • Season Grid — full table. Yellow rows = still need Hardest Worker.
📊 Season Stats
  • Unlocks after first completed game
  • Cumulative minutes, goals, assists per player

🗑 Clearing Game Data

Setup tab → Danger Zone → Clear Game Data. Two-step confirmation. Clears lineup, events, and score — keeps opponent name and date.
